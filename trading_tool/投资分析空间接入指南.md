## 一、这个项目是什么

**藤本茂融合交易策略 Web 分析工具**（`trading_tool`），一套基于藤本茂交易哲学 + 神奇九转 + Fibonacci + MA 的行情分析工具，包含：

- **数据源**：美股 / ETF / 指数（Yahoo Finance 为主，限流自动降级 Nasdaq 兜底）+ A 股（新浪财经）
- **策略引擎**：神奇九转、藤本茂阶梯仓位管理、Fibonacci、均线（MA）
- **自选看板**：一键查看全部关注标的的状态 / 操盘建议 / 九转 / 新高新低 / 近 5 日涨跌（后台刷新，接口秒回）
- **分析界面**：K 线 + 均线 + Fibonacci + 策略信号可视化（纯前端 Canvas，无需图表库）

## 三、在本空间运行

**前置条件**
- Python 3.10+（推荐 3.11）
- 机器能访问外网（美股走 Yahoo、A 股走新浪，均需联网）

**方式 A：直接运行（最简）**

```bash
cd trading_tool
pip install -r requirements.txt      # 安装依赖
python web_app.py                    # 启动（或 ./start.sh / start.bat）
# 浏览器打开 http://localhost:8000
```

自定义端口 / 监听地址：
```bash
PORT=8080 python web_app.py
HOST=127.0.0.1 PORT=8080 python web_app.py   # 仅本机
```

**方式 B：Docker（推荐服务器长期运行）**

```bash
docker compose up -d --build
# 访问 http://<本机或服务器IP>:8000
```

**方式 C：后台常驻（无 Docker 时）**

```bash
nohup python web_app.py > app.log 2>&1 &
```

## 四、数据来源与注意事项

| 市场 | 数据源 | 说明 |
|------|--------|------|
| 美股 / ETF / 指数 | Yahoo Finance（主）+ Nasdaq（兜底） | Yahoo 被限流时**全局熔断**自动切换 Nasdaq，无需逐只处理 |
| A 股 | 新浪财经 K 线 | 部分 ETF 前缀特殊，已做 `sh`/`sz` 自动换市场重试 |

要点：
- **限流是系统性处理的**：一旦 Yahoo 出现 429 / 非 200 / 连接异常，触发全局冷却，期间所有美股直接走 Nasdaq 兜底，不影响整体可用性。
- **ETF 说明**：Nasdaq 免费接口只覆盖个股、不覆盖 ETF。在 Yahoo 被封的环境里，个别纯 ETF（如 SMH / VGT / JEPI）可能取不到；**在你自有 IP / 容器部署时 Yahoo 直连正常，全部标的都能获取**。
- **自选看板后台刷新**：接口永远秒回（命中缓存直接返回，缺失/过期则后台计算），冷启动不再阻塞。
