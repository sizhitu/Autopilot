## 迁移到「投资分析」空间的步骤

在 WorkBuddy App → 项目空间「投资分析」→ 任务页 → 选择导入项目 / 关联 Git 仓库：

```
https://github.com/sizhitu/Autopilot
```

导入后，在空间内运行：

```bash
cd trading_tool
pip install -r requirements.txt
python web_app.py
# 浏览器打开 http://localhost:8000
```
